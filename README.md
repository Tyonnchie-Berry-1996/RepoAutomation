# Repo Automation

This repo is as literal as the title; two bash scripts that automate a git fetch call and a git pull call inside of your git repos

>Bash script automation

Repo Automation is made for DevEnv (container): https://github.com/Tyonnchie-Berry-1996/DevEnv

If you want to try out repo automation for your host machine go here: https://github.com/Tyonnchie-Berry-1996/Git-Automation
